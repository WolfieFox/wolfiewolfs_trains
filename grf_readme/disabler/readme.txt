Wolfie's Deduplicating Disabler
========================================

I'm kind of a stickler and I want to change things
inside timberwolf's set, but it's a little annoying
when you start having multiple copies of the same
vehicle, so this set will disable vehicles in
Timberwolf's set that are also present in my
addon set.
